# copilot-sdk

A GitHub Copilot SDK wrapper around the AG-UI protocol.

## What is this?

This package wraps the GitHub Copilot SDK to expose it through the AG-UI protocol interface, enabling agentic interactions with Copilot models.

## Getting Started

```bash
pnpm install
pnpm dev
```

## Dependencies

- `@github/copilot-sdk` - GitHub's official Copilot SDK
- `@ag-ui/core` - AG-UI protocol implementation
- `@ag-ui/client` - AG-UI client utilities
